Étape 1 : Préparer le paquet

Dix piles de cartes apparaîtront à l'écran. Seule la carte du haut de chaque pile sera face visible, tandis que les autres resteront cachées jusqu'à ce que vous les dévoiliez.

Étape 2 : Déplacez vos cartes

Trouver une carte qui est exactement une valeur inférieure à une autre (par exemple, placez un 7 sur un 8). Vous pouvez empiler des cartes de n'importe quelle couleur, mais seules les séquences dans la même couleur disparaîtront une fois complètes.

Étape 3 : Terminez une séquence de couleur

Construisez toutes les cartes du Roi jusqu'à l'As dans la même couleur. Une fois qu'une séquence complète est terminée, elle disparaîtra, vous donnant plus d'espace pour continuer à jouer.

Étape 4 : Distribuez des cartes supplémentaires

Si vous n'avez plus de mouvements, appuyez sur le bouton "Distribuer" pour mettre une nouvelle carte sur chaque pile. Essayez de laisser au moins une pile vide avant de distribuer les cartes—cela aide à garder vos options ouvertes !

Étape 5 : Gagnez le jeu

Continuez à empiler, à clarifier et à distribuer jusqu'à ce que toutes les huit séquences de couleur soient complètes. Lorsque chaque carte a été retirée, vous avez gagné Spider Solitaire !
