⚡Cómo jugar Liquid Sort

¡Comenzar a jugar Liquid Sort es tan fácil como verter agua! Verás un montón de tubos de ensayo, cada uno sosteniendo una mezcla de líquidos coloridos. Tu trabajo es mover los líquidos para que cada tubo contenga solo un color.

1. Busca los mismos colores

Toca un tubo para recoger su capa superior de líquido, luego toca otro tubo para verterlo. Solo puedes verter si los colores coinciden o si el tubo de destino está vacío y tiene espacio.

2. Usa los tubos vacíos sabiamente

Los tubos vacíos son tus ayudantes. Mantén al menos un tubo libre para poder mezclar líquidos. Este pequeño truco te ayuda a evitar quedarte atascado.

3. Planifica algunos movimientos por adelantado

Antes de verter, piensa: “¿Qué color debería ir primero?” Esto te salvará de mezclas desordenadas y callejones sin salida.

4. Apila ordenadamente para completar niveles

¡Cuando todos los tubos contengan solo un color, has superado el nivel! El rompecabezas se resuelve y se desbloquea un nuevo nivel más complicado.

5. Consejos para pequeños maestros de rompecabezas

• Comienza con colores que aparecen menos a menudo—esto crea espacio.
• No viertas si no hay un color que coincida debajo; podrías bloquear un movimiento.
• ¡Tómate tu tiempo y disfruta de las coloridas pilas!
