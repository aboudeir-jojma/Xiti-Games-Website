⚡Como Jogar Liquid Sort

Começar com Liquid Sort é tão fácil quanto derramar água! Você verá várias provetas, cada uma segurando uma mistura de líquidos coloridos. Seu trabalho é mover os líquidos para que cada tubo contenha apenas uma cor.

1. Procure as Cores Iguais

Toque em um tubo para levantar a camada superior do líquido, depois toque em outro tubo para despejá-lo. Você pode despejar apenas se as cores combinarem ou se o tubo de destino estiver vazio e tiver espaço.

2. Use os Tubos Vazios com Sabedoria

Tubos vazios são seus ajudantes. Mantenha pelo menos um tubo livre assim você pode misturar os líquidos. Esse pequeno truque ajuda você a evitar ficar preso.

3. Planeje Alguns Movimentos à Frente

Antes de despejar, pense: “Qual cor deve ir primeiro?” Isso vai te salvar de misturas bagunçadas e becos sem saída.

4. Empilhe com Cuidado para Limpar os Níveis

Quando todos os tubos contêm apenas uma cor, você venceu o nível! O quebra-cabeça se limpa, e um novo nível mais desafiador é desbloqueado.

5. Dicas para Pequenos Mestres dos Quebra-Cabeças

• Comece com as cores que aparecem menos frequentemente—isso cria espaço.
• Não despeje se não houver cor combinando embaixo; você pode bloquear um movimento.
• Leve seu tempo e aproveite as pilhas coloridas!
